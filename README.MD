# 影片解析

---

1. 首先配置代理，因为元数据网站在国外已经被墙。

> ```properties
> #http代理设置
> #是否启用代理
> https.proxy.enable=true 
> #代理IP
> https.proxy.host=127.0.0.1
> #代理端口
> https.proxy.port=8080
> ```

2. 配置超时时间，如果延迟比较严重可以将超时时间调大，注意是毫秒。

> ```properties
> #超时时间毫秒
> jsoup.timeout=5000
> ```
